Bearbeitung der Fotos

Neben der "Sterne" Bewertung, ist aktuell einzig das Ändern der Tags möglich.

Ändern der Tags eines einzelnen Bildes

  • Ein Klick auf den Stift-Button öffnet das Fenster zum Ändern der Tags

  • Füge hinzu, entferne und speichere mittels OK Button

Ändern der Tags mehrerer Bilder

Variante "einzelne Fotos auswählen"

  • Benutze die Filter um die Auswahl einzuschränken

  • Klicke auf jeden Häkchen-Button derer Fotos, dessen Tags geändert werden sollen

  • Benutze "Photos bearbeiten" aus dem Seitenmenü

  • Füge hinzu, entferne und speichere mittels OK Button

Variante "mehrere Fotos auswählen"

  • Benutze die Filter um die Auswahl einzuschränken

  • Benutze "Alle auswählen" aus dem Seitenmenü

  • Deselektiere ggf. ungewünschte Fotos

  • Oder selektiere ungewünschte Fotos und benutze dann "Auswahl umschalten aus dem Seitenmenü"

  • Benutze "Fotos bearbeiten" aus dem Seitenmenü

  • Das Fenster zum Ändern der Tags öffnet sich
